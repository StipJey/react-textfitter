react-textfitter
=========================

Work only with single line

## Installation
```bash
npm install react-textfitter --save
```

## Usage

```javascript
import { Textfit } from 'react-textfitter';

class App extends Component {
  render() {
    return (
      <Textfit
        maxFontSize={60},
        width={100},
      >
        any text
      </Textfit>
    );
  }
}
```
